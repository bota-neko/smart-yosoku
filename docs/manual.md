# smart-yosoku 使い方ガイド（説明書）

毎日の納品を記録して、明日つくる数をらくに決めるためのサービスです。
このガイドは、はじめての方でも今日から使えるように書いています。

> このサービスは、**「どのお店に・何を・いくつ卸したか」** を毎日記録すると、
> **「明日いくつ作ればいいか」** を自動で計算してくれます。むずかしい設定は後回しでOK。

## まず覚える3ステップ
やることは、これだけです。

1. **記録する** — 今日、お店ごとに卸した数を入れる（メニュー「納品入力」）
2. **見る** — 明日いくつ作ればいいかを確認（メニュー「製造計画」）
3. **ととのえる（最初だけ）** — お店と商品を登録（「卸先管理」「商品管理」）。一度やれば、あとは 1 と 2 のくり返し。

> **ポイント:** はじめは見本データが入っているので、登録をとばして「製造計画」を見るだけでも動きを試せます。

---

## 毎日の使い方
この2つを回すだけ。数分で終わります。

### 1. 今日の納品を記録する（メニュー「納品入力」）
- **日付**を選ぶ（ふだんは「今日」のまま）
- **お店（卸先）**のタブを選ぶ
- 商品ごとに**卸した数**を入力する
- ラクをするなら**「前日をコピー」**を押して数字を直すだけ
- 入れ終わったら**「この日の納品を保存」**

空欄は「まだ入れていない（未入力）」、**0**は「今日はゼロだった」という意味で区別されます。
ここで入れた数字が、そのまま予測の元になります。

### 2. 明日いくつ作るかを見る（メニュー「製造計画（合計予測）」）
商品ごとに、お店別の「卸す数」と、その合計＝**作るべき総数**が出ます。
大きな赤い数字が「明日、用意することをおすすめする数」です。「予測根拠」を押すと理由が見られます。

## 最初の設定（一度だけ）
お店と商品を登録します。むずかしい設定は、必要になってからで大丈夫です。

### お店（卸先）を登録する（メニュー「卸先管理」）
- 「お店の名前」を入れて**「追加する」**。種別（スーパー・飲食店など）は任意。
- 編集・削除・並びは後からいつでも変えられます。

### 商品を登録する（メニュー「商品管理」）
- 「商品名」と「単位（丁・パック・kgなど）」を入れて**「追加する」**。
- ↑↓ボタンで**並び替え**できます（この順番が入力画面にも反映されます）。

### くわしい設定（使い込む人だけ）
次の3つは**設定しなくても動きます**。慣れてきたら「卸先管理」の各お店を開いて調整してください。
- **取扱商品**: そのお店に卸す商品だけに絞る（例：食堂には厚揚げだけ）
- **ケース単位で卸す**: 数をケースの倍数へ切り上げる（例：厚揚げ 1ケース20パック）
- **余裕（安全率）**: 予測より何%多めに作るか（初期は10%）

## 推奨数はどう決まる？
むずかしい計算は自動です。考え方だけ知っておくと安心です。

```
売れそうな数 ＋ 余裕分 － 今ある在庫 － すでに手配した数 ＝ 卸す数（推奨）
```

- 「売れそうな数」は、これまでの納品実績（曜日ぐせ・季節・直近の伸び）から計算します。
- データがたまるほど、季節や前年との比較まで反映され、当たりやすくなります。
- 新しいお店・商品は、実績がまだ無いので最初は「参考値」と出ます。使ううちに確かになります。

## よくある質問

**Q. 入力した数字は、どこへいくの？**
そのまま「過去の納品実績」として保存され、次の日からの予測に使われます。入力＝記録＝予測の元、すべて同じデータです。

**Q. 「0」と「空欄」はどう違うの？**
空欄＝まだ入力していない（未入力）。0＝その日は卸しがゼロだった、という記録です。ゼロの日はきちんと 0 と入れてください。

**Q. 新しく登録したお店・商品の予測が「0（参考値）」なのはなぜ？**
過去の実績がまだ無いためです。数日〜数週間、納品を記録していくと、その分だけ予測が具体的になり精度も上がります。

**Q. お店によって、卸す商品や数え方を変えたい**
「卸先管理」で各お店を開き、取扱商品・ケース単位で卸す・余裕（安全率）を設定できます。設定しなければ全商品・端数OK・余裕10%で動きます。

**Q. データはどこに保存されるの？**
お試し版では、このパソコン（ブラウザ）の中に保存されます。本番運用では、組織ごとに安全に分けられたデータベース（Supabase）に保存され、他の会社からは見えません。

## 困ったとき
- **画面の見た目が崩れる** → ページを再読み込み（Mac は ⌘+Shift+R、Windows は Ctrl+Shift+R）。
- **数字が思ったより少ない／多い** → 直近の納品実績を確認。極端な日があると予測が引っぱられます。「予測詳細」で理由を確認できます。
- **予測が「参考値」のまま** → データがまだ少ない合図。記録を続けると精度が上がります。
- **まちがえて入力した** → その日付・お店をもう一度開いて直し、「保存」で上書きできます。
